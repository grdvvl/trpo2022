### Язык разметки MarkDown

**Параграф**  
aвтоматически __переносится__ на ~~следующую~~ строку
*Параграф 2*

_Параграф 3_

#### Нумерованный список

1. Первый пункт.
   1. подпункт 1.
   1. подпункт 2.
1. Второй пункт.

   Вложенный текст
   
3. Третий пункт.

#### Маркерованный список

* Первый пункт
   * вложенный подпункт 
      * вложенный во вложенный подпункт
         * вложенный во вложенный во вложенный подпункт  
- Второй пункт
+ Третий пункт

```c#
   MyClass myClass = new();
```
